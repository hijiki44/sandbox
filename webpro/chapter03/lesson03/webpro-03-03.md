# ショートカット
- テキストツール： T
- テキスト入力の確定： Command + Return
- テキスト入力のキャンセル： Esc
- 文字間のカーニング： Option + ← または Option + →
- 単語の選択： ダブルクリック
- テキスト全体の選択： トリプルクリック

# Word の行間と Photoshop の行送りの違い
- 行間： 前の段落の底辺と次の段落の天井の間のサイズ
- 行送り： 前の段落の底辺と次の段落の底辺の間のサイズ
