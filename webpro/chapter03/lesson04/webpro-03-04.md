# ショートカット
- 全体を選択する： Command + A
- 選択範囲を解除する： Command + D
- 選択したオブジェクトを複製する： Option を押しながらドラッグ
- 1 つ下のレイヤーを選択する： Option + [
- 1 つ上のレイヤーを選択する： Option + ]
- ファイルを開く： Command + O
